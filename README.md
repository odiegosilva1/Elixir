# 🚀 Estudos de Desenvolvimento Web com Elixir e Phoenix 🛠️

Bem-vindo ao repositório de estudos sobre desenvolvimento web usando **Elixir** e o framework **Phoenix**! Aqui, você encontrará recursos, exemplos de código e projetos práticos para aprender a construir aplicações web modernas e escaláveis com essas tecnologias incríveis.

## 📚 Sobre o Projeto

Este repositório foi criado para compartilhar meus estudos e aprendizados sobre **Elixir** e **Phoenix**. O objetivo é explorar conceitos como:

- 🧩 **Concorrência e Paralelismo** com Elixir
- 🌐 **Desenvolvimento Web** com Phoenix
- 🛠️ **APIs RESTful** e **WebSockets**
- 🧪 **Testes Automatizados**
- 🚀 **Deploy** de aplicações Phoenix

## 🛠️ Tecnologias Utilizadas

- **Elixir** 💧: Uma linguagem funcional moderna, construída para escalabilidade e manutenibilidade.
- **Phoenix** 🔥: Um framework web produtivo e de alta performance para Elixir.
- **Ecto** 📊: Uma ferramenta de banco de dados e mapeamento para Elixir.
- **PostgreSQL** 🐘: Banco de dados relacional utilizado nos exemplos.
- **HTML/CSS/JavaScript** 🎨: Para o front-end das aplicações.

## 📂 Estrutura do Repositório

- **/exemplos**: Códigos de exemplo para entender conceitos básicos e avançados.
- **/projetos**: Projetos completos para praticar o desenvolvimento web com Phoenix.
- **/recursos**: Links úteis, artigos e documentação para aprofundar seus conhecimentos.
- **/desafios**: Desafios práticos para testar suas habilidades.

## 🚀 Como Começar

1. **Clone o repositório**:
   ```bash
   git clone https://github.com/odiegosilva1/Elixir.git
   ```

2. **Instale as dependências**:
   ```bash
   mix deps.get
   ```

3. **Configure o banco de dados**:
   - Certifique-se de ter o PostgreSQL instalado e configurado.
   - Crie o banco de dados e execute as migrações:
     ```bash
     mix ecto.create
     mix ecto.migrate
     ```

4. **Inicie o servidor Phoenix**:
   ```bash
   mix phx.server
   ```

5. Acesse a aplicação no navegador: `http://localhost:4000`.
